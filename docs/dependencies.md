# Dependencies

## Required

- bash
- git
- curl
- sed
- awk
- sha256sum
- find
- sort
- xargs
- xdg-open
- setsid
- gum
- fzf
- bat or batcat
- eza
- nvim
- wl-copy

Note: Python is not required for update checks; `regex_help` examples mention
Python but it is not needed to run CBC.

## Optional

- zellij
- sudo
