# Dependencies

## Required

- bash
- git
- curl
- sed
- awk
- find
- xargs
- man
- xdg-open
- setsid
- Python 3.10 or newer
- fzf
- bat or batcat
- eza
- imv-x11
- nvim
- wl-copy

## Strongly Recommended

- gum (required when `CBC_USE_GUM=true`)

## Documentation Builds

- pip, installed inside the Python virtual environment
- Zensical, installed from `requirements-docs.txt`
